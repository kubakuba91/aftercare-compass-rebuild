# Phase 1 Product Definition

This folder contains the Phase 1 planning artifacts for the Aftercare Placement Platform. It translates the PRD into a build-ready v1 scope.

## Locked Decisions

- Beta focus: aftercare supply first.
- MVP depth: referral-ready.
- Authentication: Clerk only.
- Supabase: database, storage, and realtime only.
- Reviews and ratings: excluded from v1.
- Public lead capture: generic contact forms create internal lead records.
- Address privacy: exact addresses are never shown to public users or referents.
- Continued Care availability: profiles use an accepting-new-patients toggle.
- User organization membership: one email address belongs to one organization.
- Enterprise billing: Stripe custom pricing.
- Video support: URL-only in v1.
